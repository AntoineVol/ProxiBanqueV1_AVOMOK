======================Lancement de l'application==============
Pré-requis : 
* Avoir telecharger le fichier ProxiBanqueV1_AVOMOK.jar
* Posseder le JRE 10.0.1 sur votre système d'exploitation
* Avoir parametrer la variable d'environnement de java
Pour plus d'information, rendez-vous sur le site d'oracle

1 - Dans votre console, placer vous dans le dossier contenant le fichier ProxiBanqueV1_AVOMOK.jar

2 - Entrer la commande >java -jar ProxiBanqueV1_AVOMOK.jar

===============Fonctionnalitées présentes===============
Créer, modifier, supprimer un client
Créer un compte bancaire
Consulter la liste des clients et de leurs compte
Effectuer un virement entre deux comptes
Effectuer une simulation de crédit
Effectuer un placement en bourse
Auditer l'agence

=========Fonctionnalitées absente de cette version============
Modifier un compte bancaire
Consulter la liste des placements en bourse
